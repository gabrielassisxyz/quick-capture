    ```python
    """Karakeep HTTP client — send Reference-classified captures as bookmarks."""

    from __future__ import annotations

    import logging
    import os
    import re
    from typing import Any

    import httpx

    KARAKEEP_API_URL = os.getenv("KARAKEEP_API_URL", "https://karakeep.assislab.duckdns.org")
    KARAKEEP_API_KEY = os.getenv("KARAKEEP_API_KEY", "")

    _URL_PATTERN = re.compile(r"https?://\S+")

    logger = logging.getLogger(__name__)
    ```

    2. Implement `dispatch_reference_to_karakeep(text, enriched_text, tags=None, api_url=None, api_key=None) -> dict[str, Any]`:
       - Validate api_key (fallback to KARAKEEP_API_KEY env var) — raise ValueError("KARAKEEP_API_KEY not configured") if empty
       - Auto-detect URLs in text using `_URL_PATTERN.search(text)`:
         - If URL found: `type="link"`, `url=url_match.group(0)`
         - If no URL: `type="text"`, `text=text`
       - Always include: `title=text[:200]`, `note=enriched_text`, `source="api"`
       - Include tags in payload if provided: `tags=tags`
       - Send POST to `{url}/api/v1/bookmarks` with Bearer token auth, timeout=30.0
       - Call `response.raise_for_status()` — raises httpx.HTTPStatusError on failure
       - Return `response.json()`
       - NEVER include API key in logged error messages (T-2-02 mitigation)

    3. Implement `sync_reference_to_karakeep(capture_id, enrichment, conn=None, api_url=None, api_key=None) -> dict[str, Any] | None`:
       - Wrap dispatch_reference_to_karakeep in try/except
       - Parse tags from JSON string if needed (`json.loads(enrichment["tags"])` if string)
       - On success: call `log_sync(capture_id, "karakeep", conn=conn)` and return result
       - On httpx.HTTPStatusError, httpx.ConnectError, or ValueError: log warning (without API key!), return None
       - This is the graceful degradation pattern from PATTERNS.md — wiki sync continues even if Karakeep fails

       - `test_dispatch_includes_tags`: Pass tags=["programming", "rust"], verify they appear in json payload
