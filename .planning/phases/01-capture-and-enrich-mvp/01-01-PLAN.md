    **1. Create `src/quick_capture/cli.py`:**

    This is the main TUI entry point. It uses prompt_toolkit for multiline input and Rich for rendering.

    ```python
    """Quick Capture CLI — floating terminal inbox capture."""
    import sys
    from prompt_toolkit import prompt
    from prompt_toolkit.key_binding import KeyBindings
    from rich.console import Console
    from rich.panel import Panel

    from quick_capture.db import save_capture

    console = Console()

    MAX_CAPTURE_SIZE = 10_000  # 10KB limit per security threat model

    def run_capture_tui() -> str | None:
        """Display capture UI and return multiline text, or None on cancel."""
        console.print(Panel(
            "[bold cyan]Quick Capture[/bold cyan]\n"
            "[dim]Type your thought. Ctrl+S to save. Escape to cancel.[/dim]",
            border_style="cyan",
            padding=(0, 1),
        ))

        kb = KeyBindings()

        @kb.add("c-s")
        def _submit(event):
            event.current_buffer.validate_and_handle()

        try:
            text = prompt(
                "💭 ",
                multiline=True,
                key_bindings=kb,
                mouse_support=True,
                prompt_continuation=lambda width, line_number, wrap_count: "... ",
            )
            return text.strip() if text.strip() else None
        except KeyboardInterrupt:
            return None

    def main() -> None:
        """Entry point: run TUI, save to DB, exit."""
        try:
            text = run_capture_tui()
            if text:
                if len(text) > MAX_CAPTURE_SIZE:
                    console.print(f"[red]✗ Capture too large (max {MAX_CAPTURE_SIZE} chars)[/red]")
                    sys.exit(1)
                capture_id = save_capture(text)
                console.print(f"[green]✓ Saved[/green] (id: {capture_id[:8]}...)")
                sys.exit(0)
            else:
                console.print("[dim]Cancelled.[/dim]")
                sys.exit(0)
        except Exception as e:
            console.print(f"[red]✗ Failed to save[/red] — check nexus.db")
            sys.exit(1)

    if __name__ == "__main__":
        main()
    ```

    Key implementation details:
    - `Ctrl+S` triggers `validate_and_handle()` which submits the prompt
    - `Escape` raises `KeyboardInterrupt` which is caught, returning None
    - `Enter` inserts a newline (multiline=True)
    - Text is stripped; empty/whitespace-only returns None (no save)
    - `MAX_CAPTURE_SIZE = 10_000` enforced before DB write (DoS mitigation per threat model)
    - `sys.exit(0)` called explicitly after save AND after cancel to ensure terminal closes (per Pitfall 4)
    - Exception handler prints error and exits with code 1
    - Import `save_capture` from the db module created in Task 1

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| user → TUI input | Untrusted text input from keyboard (up to 10KB) |
| TUI → SQLite (nexus.db) | Trusted application writing to shared database |
| Python process → ghostty terminal | Terminal closes via sys.exit() |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-01 | Tampering | db.py | mitigate | All SQL queries use parameterized `?` placeholders — never string concatenation |
| T-01-02 | Tampering, Elevation | cli.py → db.save_capture | mitigate | Validate text length ≤ 10KB before DB write (DoS mitigation); sanitize by storing raw text without interpretation |
| T-01-03 | Tampering | db.py → nexus.db | mitigate | Path.resolve() on db_path to prevent path traversal; NEXUS_DB is hardcoded constant, never from user input |
| T-01-04 | Denial | cli.py | mitigate | MAX_CAPTURE_SIZE = 10_000 character limit enforced in main() before DB write |
| T-01-05 | Tampering | cli.py | accept | TUI input is not interpreted as SQL or commands — it's stored as plain text only |
</threat_model>
